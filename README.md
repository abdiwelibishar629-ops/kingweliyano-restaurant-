# 🍽️ King Weliyano Restaurant - Ordering System

A complete restaurant ordering system with customer ordering, waiter dashboard, chef dashboard, and admin panel.

---

## 🚀 Live Demo

**Customer Page:** https://king-weliyano-restaurant.onrender.com/

---

## 📋 Features

### 👤 **Customer Side** (No Login Required)
- Browse menu with categories
- Add items to cart
- Place orders with name and phone
- Real-time order confirmation

### 👨‍🍳 **Waiter Dashboard** (Login Required)
- View all orders
- Filter orders by status
- Update order status (pending → preparing → ready → served)

### 👨‍🍳 **Chef Dashboard** (Login Required)
- View pending orders
- Start cooking orders
- Mark orders as ready

### 👑 **Admin Dashboard** (Login Required)
- View order statistics
- Manage menu items (add/edit/delete)
- View all orders
- Delete orders
